## 🌐 الموديول 1: مراقب حركة الشبكة في الوقت الحقيقي

### محرك الالتقاط الأساسي
- **التقاط الحزم الخام (Raw Socket)** — يستخدم مقابس Windows الخام `SIO_RCVALL` لمراقبة حقيقية على مستوى الحزم
- **الدعم المزدوج IPv4/IPv6** — التقاط متزامن على مقبسي IPv4 و IPv6 في نفس الوقت
- **اكتشاف كرت الشبكة الذكي** — خوارزمية ذكية تستعلم جدول التوجيه (`GetBestInterface`) لاختيار كرت الإنترنت النشط تلقائياً
- **تحديث تلقائي لكروت الشبكة** — الضغط على القائمة المنسدلة يكتشف فوراً أي اتصال واي فاي/إيثرنت/VPN جديد
- **تصفية الكروت الوهمية** — استبعاد تلقائي لـ VMware, VirtualBox, Radmin, ZeroTier, والكروت الافتراضية الأخرى
- **خيوط عالية الأداء** — خيط التقاط مخصص بأولوية عالية مع مخزن مؤقت 32 ميجابايت
- **المعالجة الجماعية** — تجميع الحزم (250 مللي ثانية أو 3000 حزمة) لتقليل الحمل على واجهة المستخدم
- **إيقاف قسري للالتقاط** — إيقاف سلس مع مهلة ثانيتين، ينتقل لإنهاء قسري للخيط إذا لم يستجب
- **استعادة صامتة من أخطاء الـ Socket** — إعادة تشغيل تلقائية للبرنامج في حال خطأ ربط المقبس

### تتبع حركة المرور لكل عملية
- **تحليل PID في الوقت الحقيقي** — ربط كل حزمة بالعملية المالكة لها عبر `GetExtendedTcpTable` / `GetExtendedUdpTable`
- **ربط عمليات IPv6** — دعم كامل لجداول TCP/UDP الخاصة بـ IPv6
- **تحليل أسماء الخدمات** — التعرف على خدمات Windows العاملة تحت `svchost.exe` عبر `EnumServicesStatusEx`
- **تخزين مؤقت ذكي لحالة TCP** — يتعامل مع حالات LISTEN و ESTABLISHED وربط المنافذ العامة (0.0.0.0)
- **خيط تخزين مؤقت في الخلفية** — خيط منفصل يحدّث ذاكرة التخزين المؤقت كل 1.5 ثانية

### شجرة عرض الشبكة
- **نموذج بيانات مخصص QAbstractItemModel** — نموذج شجري هرمي: عقد العمليات ← أبناء الاتصالات
- **8 أعمدة بيانات** — التطبيق/البروتوكول، المصدر، الوجهة، الخدمة، سرعة التنزيل، سرعة الرفع، إجمالي البايتات، الحزم
- **حساب السرعة الحية** — تحديث لكل اتصال ولكل عملية كل ثانية
- **الفرز الحي** — فرز رقمي دقيق عبر `UserRole+5` و `QSortFilterProxyModel`
- **البحث/التصفية** — تصفية تكرارية فورية عبر جميع الأعمدة
- **توسيع/طي الكل** — زر تبديل لتوسيع الشجرة
- **أيقونات العمليات** — استخراج أيقونات التطبيقات الحقيقية عبر `SHGetFileInfoW`
- **صلاحيات التصحيح** — تفعيل `SeDebugPrivilege` للوصول لأيقونات عمليات النظام

### القائمة المنسدلة (كليك يمين)
- **إنهاء العملية** — إنهاء إجباري لأي عملية عبر PID
- **نافذة خصائص العملية** — حوار احترافي يعرض:
  - أيقونة العملية بحجم 150×150
  - اسم العملية، PID، النوع (تطبيق/خدمة/نظام)
  - المسار الكامل للملف (نص قابل للتحديد)
  - إحصائيات الترافيك: سرعة التنزيل/الرفع، الاستهلاك الكلي، عدد الحزم
  - زر "فتح موقع الملف" (يفتح المستكشف مع تحديد الملف)
- **نسخ تفاصيل الاتصال** — نسخ معلومات المصدر:المنفذ للحافظة

### بطاقات لوحة المعلومات
- **عداد الحزم** — إجمالي الحزم الملتقطة
- **سرعة التنزيل** — سرعة RX فورية مع تدرج تلقائي (B/s → KB/s → MB/s)
- **سرعة الرفع** — سرعة TX فورية مع تدرج تلقائي
- **سرعة الهاردوير** — السرعة على مستوى النظام عبر عدادات كرت الشبكة الحقيقية (`GetIfEntry2`)

### البروتوكولات المدعومة
- TCP, UDP, ICMP, ICMPv6, TCP6, UDP6
- خريطة الخدمات: HTTP, HTTPS, DNS, SSH, FTP, SMTP, IMAP, POP3, MySQL, PostgreSQL, Redis, MongoDB, RDP, VNC, NTP, NetBIOS, SMB, DHCP, IKE, OpenVPN, والمزيد

## 🛡️ الموديول 2: مدير جدار الحماية (Firewall)

### تكامل Windows COM API
- **واجهة INetFwPolicy2 المباشرة** — تكامل COM أصلي مع جدار حماية Windows (بدون netsh)
- **مجموعة INetFwRules** — عمليات CRUD كاملة على قواعد جدار الحماية
- **بديل Netsh** — انتقال تلقائي لـ `netsh advfirewall` للحالات الخاصة

### إدارة القواعد
- **حظر التطبيق** — إنشاء قواعد حظر مزدوجة (واردة + صادرة)
- **سماح التطبيق** — إنشاء قواعد سماح مزدوجة (واردة + صادرة)
- **إزالة القواعد** — حذف القواعد بالاسم أو مسار التطبيق
- **تفعيل/تعطيل** — تفعيل أو تعطيل أي قاعدة بدون حذفها
- **تحديد متعدد** — تحديد عدة قواعد مع اختصارات Ctrl+A و Delete
- **تسمية موحدة** — نظام تسمية `NetGuard_<اسم_التطبيق>_<الإجراء>_<الاتجاه>`

### حل الاختصارات
- **دعم ملفات .lnk** — حل اختصارات Windows للمسارات الحقيقية عبر `IShellLinkW`
- **توسيع متغيرات البيئة** — معالجة `%SystemRoot%`، `%ProgramFiles%`، إلخ
- **تنقية المسار** — تحويل صارم للفواصل المائلة الخلفية لتوافق COM API

### وضع القائمة البيضاء (الإغلاق الكامل)
- **إقفال الشبكة الكامل** — حظر كل حركة المرور الصادرة عبر جميع الملفات الشخصية
- **تعطيل القواعد المتداخلة** — تعطيل جميع قواعد السماح غير التابعة لـ NetGuard
- **حماية الخدمات الحرجة** — سماح تلقائي لـ `svchost.exe` و `System` لمنع انهيار النظام
- **تبديل بنقرة واحدة** — تفعيل/تعطيل مع حوار تأكيد تحذيري

### واجهة ثنائية التبويب
- **تبويب NetGuard الأساسي** — عرض القواعد المُنشأة بواسطة البرنامج فقط
- **تبويب قواعد النظام** — متصفح كامل لقواعد جدار حماية Windows
- **مفاتيح الاتجاه** — أزرار راديو للواردة/الصادرة في كل تبويب
- **بحث/تصفية** — تصفية فورية عبر جميع الأعمدة

### السحب والإفلات
- **إسقاط ملفات EXE** — اسحب أي ملف .exe أو .lnk على تبويب الجدار الناري للحظر/السماح
- **حوار الإجراء** — اختر حظر، سماح، أو إلغاء عند الإسقاط

### الاستيراد / التصدير
- **تصدير JSON** — تصدير جميع قواعد NetGuard إلى ملف JSON
- **استيراد JSON** — استيراد وإعادة إنشاء القواعد من نسخة JSON احتياطية

### ملفات قواعد جدار الحماية الجاهزة (6 ملفات JSON)
- **🏢 الدرع العام لبيئة العمل** — للمكاتب: المتصفحات والاجتماعات المرئية والتخزين السحابي مسموح؛ التطبيقات الخلفية محظورة
- **🚀 بطولة الرياضات الإلكترونية** — للاعبين المحترفين: أقل بنق بحظر التحديثات؛ منصات الألعاب والمحادثة الصوتية والبث المباشر مسموح
- **💻 صندوق رمال المطوّرين** — لمهندسي البرمجيات: اتصال مطلق لـ IDEs و Docker وقواعد البيانات ومساعدي الذكاء الاصطناعي
- **🔒 حصن الخصوصية المطلقة** — أقصى حماية ضد التتبع: فقط متصفحات Tor/Brave وVPN والمراسلة المشفرة
- **📥 طليعة التحميل والميديا** — للأجهزة المخصصة: فقط مدراء التحميل وعملاء التورنت ومشغلات الميديا
- **🚫 جدار العزل وإيقاف التتبع** — يُجبر أدوبي وأوتوديسك وكوريل على الأوفلاين للحفاظ على الرخص

### نموذج جدول الجدار الناري
- **6 أعمدة** — التطبيق، اسم القاعدة، الإجراء، البروتوكول، الحالة، النوع
- **أيقونات التطبيقات** — `QFileIconProvider` لأيقونات التطبيقات الحقيقية
- **ترميز الألوان** — أخضر للسماح، أحمر للحظر، ذهبي لقواعد NetGuard
- **متوافق مع الثيم** — الألوان تتكيف مع الوضع الداكن/الفاتح

## 💻 الموديول 3: مراقب أجهزة النظام (Hardware)

### مصادر البيانات
- **حمل المعالج** — عداد PDH: `\Processor(_Total)\% Processor Time`
- **نشاط القرص** — عداد PDH: `\PhysicalDisk(_Total)\% Disk Time`
- **استخدام الذاكرة** — واجهة `GlobalMemoryStatusEx` (مستخدم/إجمالي/نسبة)
- **حمل كرت الشاشة** — WMI: `Win32_PerfFormattedData_GPUPerformanceCounters_GPUEngine` (محرك 3D)
- **الشبكة I/O** — واجهة `GetIfTable` (جميع الواجهات غير اللوبباك)

### مراقبة درجة الحرارة
- **حرارة المعالج** — مصادر متعددة مع بدائل تتالية:
  1. `MSAcpi_ThermalZoneTemperature` (ROOT\WMI) — تحويل من كلفن
  2. `Win32_TemperatureProbe` (CIMV2) — مئوية مباشرة
  3. `Win32_PerfFormattedData_Counters_ThermalZoneInformation` — هجين
- **حرارة القرص** — `MSFT_StorageReliabilityCounter` (ROOT\Microsoft\Windows\Storage)
- **حرارة كرت الشاشة** — استعلامات WMI لمستشعرات GPU
- **حرارة اللوحة الأم** — مستخرجة من مناطق ACPI الحرارية (أدنى قراءة مستشعر)

### بنية WMI
- **3 مساحات أسماء WMI** — `ROOT\CIMV2`، `ROOT\WMI`، `ROOT\Microsoft\Windows\Storage`
- **تحليل Variant قوي** — يتعامل مع جميع أنواع البيانات (أعداد صحيحة، عشرية، نصوص)
- **جمع بيانات مترابط** — خيط خلفي `QThread` مع فترة استطلاع 1 ثانية

### عناصر واجهة مخصصة
- **مقياس دائري (CircularGauge)** — رسم قوس دائري مخصص مع:
  - قوس القيمة (أزرق < 60%، كهرماني < 85%، أحمر ≥ 85%)
  - نص النسبة المئوية المركزي (18pt عريض)
  - نص فرعي لدرجة الحرارة
  - تسمية الجهاز
- **شريط الحرارة (TempBar)** — شريط حرارة أفقي بألوان متدرجة
- **رسم متوافق مع الثيم** — يستخدم `QPalette` للتكيف التلقائي مع الداكن/الفاتح

## 🔄 الموديول 9: المثبّت الذكي والمحدث الهوائي

### المثبّت الذكي
- **نشر تلقائي** — عند التشغيل لأول مرة، ينسخ نفسه إلى `C:\Program Files\NetGuard\UltimateNetGuard.exe`
- **رفع صلاحيات UAC** — يطلب تلقائياً صلاحية المسؤول عبر `ShellExecuteExW` (runas)
- **اختصار سطح المكتب** — ينشئ `NetGuard AIO.lnk` عبر واجهة `IShellLink` COM
- **أذونات الملف** — يضبط أذونات قراءة/كتابة/تنفيذ صارمة على الملف المثبت
- **تتبع الإصدار** — يكتب `version.dat` لاكتشاف إعادة التثبيت مقابل التحديث
- **إنهاء واستبدال** — عند التحديث، ينهي العملية القديمة قبل الكتابة فوق الملف
- **حوار تقدم التثبيت** — نافذة بدون حواف، قابلة للسحب، متوافقة مع الثيم

### المحدث الهوائي الصامت (OTA)
- **فحص خلفي** — يجلب ملف الإصدار عن بُعد بصمت بعد ثانيتين من بدء التشغيل
- **مقارنة إصدارات ذكية** — مقارنة `major.minor.patch` ذكية، يعمل فقط عندما الإصدار البعيد أعلى حصراً
- **حوار التحديث** — نافذة بدون حواف متوافقة مع الثيم تعرض الإصدار الجديد وسجل التغييرات
- **توجيه للموقع** — زر "حدّث الآن" يفتح صفحة التحميل الرسمية على موقع المطور

## 🔒 ميزات الأمان والنظام

- **رفع صلاحيات المسؤول** — مانيفست UAC مع `requireAdministrator`
- **نسخة واحدة فقط (QSharedMemory)** — حارس `QSharedMemory` يمنع تشغيل نسخ متعددة؛ إذا شُغلت نسخة ثانية، يجد النافذة الموجودة عبر `FindWindowW` ويركز عليها
- **غير مدرك لـ DPI** — تخطيط متسق عبر جميع عوامل التكبير (DWM يتولى التحجيم)
- **توافق Windows 7-11** — إدخالات `supportedOS` كاملة في المانيفست
- **قاعدة جدار ناري تلقائية** — إنشاء تلقائي لقاعدة سماح واردة لنفسه
- **تصغير لعلبة النظام** — زر الإغلاق يصغر لعلبة النظام (لا يخرج)
- **إشعارات Windows مخصصة** — رسائل فقاعية في علبة النظام
- **تشغيل تلقائي مع النظام** — تسجيل اختياري في `HKCU\...\Run` للتشغيل عند بدء الويندوز
