### Purpose Statement

**Ultimate NetGuard AIO** is a **legitimate network monitoring and system administration tool** designed for use by network administrators, security professionals, IT support teams, and power users on systems they own or manage.

This software is functionally equivalent to established, widely-accepted tools such as:
- Windows Resource Monitor (built into Windows)
- Windows Performance Monitor (built into Windows)
- Microsoft Network Monitor
- Sysinternals Process Monitor
- Windows Firewall with Advanced Security (GUI wrapper)

### Intended Use

This software is intended **EXCLUSIVELY** for:

1. ✅ Monitoring network traffic on computers you **own**
2. ✅ Monitoring network traffic on computers you have **explicit written authorization** to monitor
3. ✅ Managing Windows Firewall rules on systems under your **direct administrative control**
4. ✅ System administration and IT support within your organization
5. ✅ Educational purposes for learning about networking, protocols, and Windows APIs
6. ✅ Debugging network connectivity issues on your own machine

### Technical Transparency

To ensure full transparency about the software's capabilities:

1. **What this software CAN see:**
   - IP addresses (source and destination)
   - Port numbers (source and destination)
   - Protocol types (TCP, UDP, ICMP, etc.)
   - Packet sizes and counts
   - Which process owns each connection
   - Network speed (bytes per second)

2. **What this software CANNOT see:**
   - Encrypted packet content (HTTPS, TLS, SSH payloads)
   - Passwords or authentication credentials
   - Email content
   - File contents being transferred
   - Website content
   - Chat messages

3. **What this software stores:**
   - Per-application bandwidth usage statistics (aggregated daily totals only)
   - User preferences (theme, overlay settings)
   - NO raw packet captures are stored
   - NO personal data is collected or transmitted
